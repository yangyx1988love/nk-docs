## 上传至gitee
*将项目加入本地git仓库
```javascript
　　git add . 
	git status #查看
```
*输入云端仓库的用户名，密码，验证成功开始上传并完成
```javascript
	git remote add origin https://gitee.com/henry-tech/nk-docs.git
　　git push --set-upstream origin master -f
```
